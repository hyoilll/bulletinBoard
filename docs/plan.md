# Laravel + MySQL + Docker 掲示板アプリ 仕様書（インデックス）

## 概要

Laravel学習のためのDB CRUD実装プロジェクト。
Docker上にLaravel 12 + MySQL + Nginxの環境を構築し、認証付き掲示板を実装する。

---

## 技術スタック

| 項目 | 採用技術 |
|---|---|
| バックエンド | Laravel 12 (PHP 8.2) |
| 認証 | Laravel Breeze (Blade) |
| データベース | MySQL 8.0 |
| Webサーバ | Nginx |
| コンテナ | Docker / Docker Compose |
| フロントエンド | Blade テンプレート + Bootstrap 5 (CDN) |

---

## 機能スコープ

### 権限マトリクス

| 操作 | ゲスト | ログインユーザー |
|---|---|---|
| 投稿一覧・詳細を見る | ○ | ○ |
| コメントを見る | ○ | ○ |
| 投稿を作成 | ✗ | ○ |
| 自分の投稿を編集・削除 | ✗ | ○（自分のみ） |
| コメントを投稿 | ✗ | ○ |
| 自分のコメントを編集・削除 | ✗ | ○（自分のみ） |
| いいねを押す | ✗ | ○ |

### 主要機能

- **投稿 CRUD**: 作成・一覧・詳細・編集・削除（自分のみ）
- **カテゴリ**: 投稿に任意付与。カテゴリ選択フィルタ時はそのカテゴリの投稿のみ表示
- **コメント CRUD**: 投稿・一覧・編集・削除（自分のみ）
- **いいね**: トグル式（同一ユーザーが同一投稿に1回のみ）

## 実装ステップ

1. **Docker インフラ構築** → コンテナ起動確認
2. **Laravel 12 + Breeze インストール** → 認証動作確認
3. **DB マイグレーション + シーダー** → phpMyAdmin で確認
4. **Model / Policy 実装** → リレーション・認可定義
5. **Controller / FormRequest 実装** → CRUD ロジック
6. **ルーティング定義** → `php artisan route:list` で確認
7. **ビュー作成** → 各画面 Blade テンプレート
8. **動作確認** → 全機能テスト
